# 🚀 shiroya.in
### Everything you need. One domain.

**[shiroya.in](https://shiroya.in)** is a premium, browser-first tools platform designed for instant productivity. We believe that professional utilities should be frictionless, private, and accessible from any device without the need for accounts or complex setups.

---

## ✨ Features

- **⚡ Instant Access**: No login, no signup, no friction. Use tools immediately in your browser.
- **🔒 Privacy First**: Your data never leaves your machine. All processing happens locally (Client-Side).
- **🚀 High Performance**: Built with Vite and React for snappy, lightning-fast interactions.
- **🛠️ Integrated Utilities**: A growing collection of tools including file sharing (Drop) and PDF processing.

---

## 🛠️ Tech Stack

- **⚡ Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **🎨 Styling**: [TailwindCSS 4](https://tailwindcss.com/) (Modern CSS Engine)
- **✨ Motion**: [Framer Motion](https://www.framer.com/motion/)
- **📍 Routing**: [React Router 7](https://reactrouter.com/)
- **📦 Fonts**: [Inter](https://rsms.me/inter/) & [Space Grotesk](https://floriankarsten.github.io/space-grotesk/)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VG31OP/shiroya.in.git
   cd shiroya.in
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🔒 Privacy note

**Your data stays with you.** 

Unlike traditional platforms, Shiroya does not store your files on a server. Every tool—from PDF merging to peer-to-peer sharing—operates locally in your browser or via encrypted direct connections. We do not track personal identities, and we do not sell data.
